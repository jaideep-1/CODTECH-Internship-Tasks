# Task 2: Emotion Classification using Machine Learning

## Dataset
Chat Emotion Dataset  
https://www.kaggle.com/datasets/nelgiriyewithana/emotions

## Objective
Build a multi-class machine learning model to classify emotions from text messages.

## Tools Used
- Python
- Pandas
- Scikit-learn
- TF-IDF
- Logistic Regression

## Steps
1. Loaded and explored emotion-labeled text data.
2. Performed feature extraction using TF-IDF.
3. Split data into training and testing sets.
4. Trained Logistic Regression classifier.
5. Evaluated model using accuracy, classification report, and confusion matrix.

## Output
- Accuracy score
- Classification report
- Confusion matrix visualization
